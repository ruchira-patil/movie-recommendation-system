🎬 Movie Recommendation System

The Movie Recommendation System is an interactive web application built using Python, Machine Learning, and Streamlit that suggests movies similar to a selected movie.

The system uses a content-based filtering approach where movie recommendations are generated based on similarities between movie features such as genres, keywords, cast, and overview.

The application integrates with the TMDB API to fetch movie posters and additional details, providing users with a visually appealing and informative interface.

This project demonstrates how to build a complete machine learning application with a user-friendly frontend using modern tools.

📱 App Preview

The application provides a clean and intuitive interface where users can:

Select a movie

🧠 How the Recommendation System Works

The system follows these steps:

1️⃣ Data Preprocessing

Movie metadata such as genres, keywords, cast, and overview are combined into a single feature called tags.

Example:

tags = overview + genres + keywords + cast + director
2️⃣ Text Vectorization

The tags are converted into numerical vectors using CountVectorizer.

This converts text data into a format suitable for machine learning models.

3️⃣ Similarity Calculation

Using Cosine Similarity, the system calculates similarity scores between movies.

similarity = cosine_similarity(vectors)

Movies with higher similarity scores are recommended.

4️⃣ Recommendation Generation

When a user selects a movie:

The movie index is found.

Similarity scores are retrieved.

Movies with the highest similarity scores are selected.

Posters and details are fetched using the TMDB API.
